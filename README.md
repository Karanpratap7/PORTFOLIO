# 💼 Portfolio Website

Personal portfolio of Karan Pratap Singh — AI Systems & Full-Stack Developer. An editorial, motion-driven single-page app showcasing projects, experience, skills, and an interactive skill chart.

🔗 **Live Website:** [karanpratap7.github.io/PORTFOLIO](https://karanpratap7.github.io/PORTFOLIO/)

## ✨ Features

- Buttery inertia smooth scroll (Lenis)
- Scroll-triggered reveals, staggered text, velocity-reactive marquee & parallax (Framer Motion)
- Magnetic buttons and a spring-follow custom cursor
- Light / dark theme toggle (press `A`), live IST clock + mouse-coordinate readout
- Interactive radar skill chart (theme-aware canvas)
- Optional UI sound feedback (WebAudio)
- Fully responsive; respects `prefers-reduced-motion`

## 🚀 Tech Stack

- React 19 + Vite
- Framer Motion (animation)
- Lenis (smooth scroll)
- Space Grotesk · Instrument Serif · Space Mono

## 🛠️ Development

```bash
npm install      # install dependencies
npm run dev      # start the dev server
npm run build    # production build to dist/
npm run preview  # preview the production build
```

## 🚢 Deployment

Deployed to GitHub Pages via GitHub Actions ([.github/workflows/deploy.yml](.github/workflows/deploy.yml)) on every push to `main`.
One-time setup: repo **Settings → Pages → Source → GitHub Actions**.
